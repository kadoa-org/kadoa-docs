## Python

Install the Python SDK using [pip](https://pypi.org/project/kadoa-sdk/):

```bash
pip install kadoa-sdk
```

### Listen to real-time changes

```python
from kadoa_sdk import Kadoa
kadoa_props = {
    "team_api_key": TEAM_API_KEY
}

kadoa_client = Kadoa(**kadoa_props)

def process_event(event):
    print(event)

kadoa_client.realtime.listen(process_event)
```

## TypeScript / JavaScript

Install the TypeScript SDK using [npm](https://www.npmjs.com/package/@kadoa/node):

```bash
npm install @kadoa/node
```

### Listen to real-time changes

```javascript
const { Kadoa } = require("@kadoa/node");

const main = () => {
  const kadoaProps = {
    teamApiKey: process.env.KADOA_TEAM_API_KEY,
  };
  const kadoaClient = new Kadoa(kadoaProps);

  function customProcessEvent(event) {
    console.log("Event received:", event);
  }

  kadoaClient.realtime.listen(customProcessEvent);
};

main();
```
